# Sample Report Outlines

These outlines are validation artifacts, not production report templates.

## Research Brief Outline

Validation targets:

- framing;
- verification;
- handoff contract.

Sections:

1. Framed objective.
2. Jurisdictions and time window.
3. Key verified developments.
4. Source table.
5. Policy-risk notes.
6. Market-risk context.
7. Competing interpretations.
8. Confidence and uncertainty.
9. Open questions.
10. Kernel validation notes.

## Return-To-Kernel Note Outline

Validation targets:

- return-to-kernel behavior;
- missing constraints;
- handoff insufficiency.

Sections:

1. Reason for return.
2. Missing fields.
3. Risk of proceeding.
4. Suggested kernel reframe.
5. Suggested user clarification questions.
