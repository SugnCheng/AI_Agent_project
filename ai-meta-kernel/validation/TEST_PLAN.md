# Test Plan

## Purpose

Validate whether the Meta-Layer produces usable handoff objects before downstream execution.

## Test Groups

1. Low-risk simple requests.
2. Ambiguous requests requiring reframe.
3. High-risk financial or policy-sensitive requests.
4. Conflicting evidence requests.
5. Downstream handoff insufficiency cases.

## Required Artifacts

- raw request;
- kernel output;
- downstream adapter response;
- validation score;
- failure notes;
- suggested kernel revisions.
