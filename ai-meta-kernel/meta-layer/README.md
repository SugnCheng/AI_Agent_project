# Meta-Layer

This directory contains the core Meta-Cognitive Kernel.

The kernel is Layer 0. It must run before downstream agents and before tool-led execution.

Primary responsibilities:

- frame the real objective;
- classify the task;
- calibrate risk;
- trigger H1-H11 habits;
- separate facts, assumptions, inferences, unknowns, constraints, stakeholders, and trade-offs;
- define required checks;
- run challenge loop;
- emit status flags;
- produce downstream handoff.

`../META_LAYER_MASTER_SPEC.md` is the canonical source of truth.
