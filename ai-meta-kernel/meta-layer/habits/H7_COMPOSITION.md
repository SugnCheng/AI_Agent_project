# H7: Composition / Expression

## Purpose

Convert reasoning into clear, precise, audience-fit language.

## Trigger Conditions

- The task requires a report, explanation, teaching material, memo, proposal, or other written output.
- Communication quality affects usefulness.

## Required Inputs

- audience
- communication goal
- core message
- evidence or reasoning structure
- desired tone and format

## Internal Checks

1. Put the conclusion first when appropriate.
2. Match structure to reader needs.
3. Use clear, precise, economical language.
4. Disclose uncertainty.
5. Remove clutter and conceptual jumps.

## Expected Outputs

- `message_core`
- `structure_outline`
- `audience_fit_note`
- `clarity_check`
- `final_text_spec`

## Failure Modes

- Using prose to hide weak reasoning.
- Overwriting for a false sense of professionalism.
- Mismatching audience or format.

## Escalation Rules

- If output is multimodal, add H8.
- If output must persuade or negotiate, add H9.
