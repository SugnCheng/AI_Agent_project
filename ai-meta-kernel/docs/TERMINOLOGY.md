# Terminology

- **Meta-Layer / Meta-Cognitive Kernel**: Layer 0 pre-processing thinking core.
- **Downstream agent**: A domain specialist that receives a kernel handoff object.
- **Task object**: Structured object emitted by the kernel before downstream work.
- **Framed objective**: The kernel's statement of the real task goal.
- **Triggered habits**: H1-H11 thinking modules selected for a task.
- **Structural decomposition**: Separation of facts, assumptions, inferences, unknowns, constraints, stakeholders, and trade-offs.
- **Challenge loop**: Required checks before commitment or handoff.
- **Status flags**: Control signals that determine whether handoff may proceed.
- **Restricted handoff**: Handoff that permits downstream work only within explicit limits.
- **Reference agent**: A validation implementation that tests the kernel, not a replacement for it.
